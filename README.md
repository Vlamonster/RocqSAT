You can run a problem from the `/code` directory using:
```bash
cat ./problems/<problem>.cnf | dune exec ./runner/runner.exe
```
The following problems can be solved within a reasonable time:
- test.cnf (SAT)
- tutorial.cnf (SAT)
- trivial.cnf (SAT)
- trivial2.cnf (UNSAT)
- simple.cnf (SAT)
- hole6.cnf (UNSAT)
- zebra.cnf (SAT)
